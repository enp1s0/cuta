# cutt - CUDA Tensor Toolkit

Cutt is a auxiliary library for tensor operations on NVIDIA GPU.

## Functionalities
- `cutt::reshape`

## Installation
```bash
git clone https://github.com/enp1s0/cutt
cd cutt
mkdir build
cd build
cmake .. # Optional: -DCMALE_INSTALL_PREFIX=/path/to/install
make -j4
# Optional: make install
```
